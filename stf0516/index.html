Why Go Open-source?

  • Lower start-up costs
  • Lower maintenance costs
  • Future-proof
  • Ability to protect your investment
  • Ability to contribute

Free as In WordPress

  • GNU General Public License, version 2
    • Source code must be available
    • All derivative code must use same license or compatible
    • The code doesn't have to be free (gratis)
  • Plugins, Themes and time still cost
